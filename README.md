# Shoirhi's Homebrew tap

Homebrew formulae for Shoirhi's tools.

## Usage

```bash
brew install Shoirhi/tap/macos-website-blocker
```

`Shoirhi/tap` is shorthand for this repository (`Shoirhi/homebrew-tap`).

## Formulae

| Formula | Description |
|---|---|
| `macos-website-blocker` | Hosts-layer website blocker for macOS ([source](https://github.com/Shoirhi/macos-website-blocker)) |

> Formulae here are updated automatically by the source project's release
> workflow. Do not edit `Formula/*.rb` by hand — edit the source of truth in the
> upstream repo instead.
